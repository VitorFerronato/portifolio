Trabalhos recentes

Easy Bills

Easy Bill é um aplicativo da web responsivo desenvolvido com animações Vue.js 3, Firebase e Vue, projetado para simplificar suas necessidades de faturamento e rastreamento de despesas. Com o Easy Bill, você pode gerenciar facilmente suas contas e despesas em um ambiente reativo e fácil de usar.

Deploy GitHub

Infinite Scroll

Este projeto é uma implementação de uma lista de rolagem infinita usando Vue.js, vueUse, dummyJson e Axios. O aplicativo busca dados fictícios da API usando dummyJson e os exibe na interface usando Vue.js, aproveitando o recurso de rolagem infinita para carregar mais itens conforme o usuário rola para baixo.

Deploy GitHub

RocketQ

O Rocket.Q é uma aplicação de interação através de perguntas, sendo possível criar uma sala para internautas anônimos fazerem perguntas e o criador da sala em posse de uma senha gerenciar essas perguntas e marcar como lidas.

GitHub

Make your burger.

O Make your burger é um aplicação feita com vue, temos o frontend que é onde o cliente preenche o formulário do hamburger, esse pedido vai para o banco de dados e retorna ao dashboard de através de uma API com todos os pedidos que podem ser alterados em um formato de CRUD.
